Это учебный проект. Проект представляет собой простое веб приложение основанное на appache tomcat версии 8. Так же веб приложение использует SQL базу для хранения некоторых переменных. Сборка проекта производиться при помощи Maven, все зависимости так же прилагаются.
Все что вам понравится вы можете использовать.
